Quidstats

Currently under active development

Technologies: HTML, CSS, Javascript, Angular, Ruby

Quidstats is a fully featured stats taking web-app centered around quidditch (yes, the game from Harry Potter. It was adapted to non-flying brooms in 2005). I started developing a now defunct mobile app for stats in 2011, but have since moved it online. I started building this app in 2015, and continue to iterate and improve it. This was my first real web development project. I think it's a pretty ambitious to start my web development career with, but I liked the idea so much I pushed through the difficult learning process. This site wouldn't exist without the help of David Brownman, who helped me get it going. I knew a bit of HTML and CSS at the start of this project. I had to brush up on that, as well as all the other techs that make this app run.

LessonBuilder

Currently under active development, but I have left the company

Technologies: HTML, CSS, Coffeescript, React.js

This is a web app I built at my last job. I'm including it here because I was the lead on the project while I was there. I was responsible for creating the app, iterating on it, adding features, and gathering and responding to feedback. This is a free tool for teachers we built as part of a research group at the University of Michigan. I had an amazing time working with the handful of developers there. Obviously, this app was a team effort. We had an existing backend that this app needed to integrate with. We also had other apps that needed to interoperate with this one. As a team, we were able to accomplish those goals and get this app (and others) in to the hands of teachers. It's a free app, if you want to check it out and play with it. All it takes is a Google Drive account.
